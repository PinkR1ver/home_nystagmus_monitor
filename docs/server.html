Server setup

Server Quick Start

A slower-paced setup page for backend deployment, uploads, and dashboard access.

← Back to landing
01

Clone repo and enter server directory

git clone <your-git-repo-url>
cd home_nystagmus_monitor/server
02

Create environment and install dependencies

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
03

Prepare algorithm runtime and checkpoint

Default path:
server/vendor/SwinUNet-VOG/
server/models/checkpoint_best.pth

Or set env:
HNM_VOG_DIR=/abs/path/SwinUNet-VOG
VOG_CHECKPOINT_PATH=/abs/path/checkpoint_best.pth
04

Start server

cd server
source .venv/bin/activate
uvicorn main:app --host 0.0.0.0 --port 8787 --reload

Endpoints: /health, /dashboard, /api/videos, /api/records.

05

Configure Android base URL

  • Emulator: http://10.0.2.2:8787
  • Physical device: http://<your-lan-ip>:8787

Set only the base URL in app settings. The upload endpoint suffix is appended by app logic.
